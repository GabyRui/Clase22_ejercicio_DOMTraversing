¿Qué es Dom Traversing?

Es la acción de navegar la estructura del DOM de un sitio web, de tal forma que se pueda interactuar con los elementos HTML. Cada elemento se representa de forma jerárquica como nodo en un árbol y el DOM TRAVERSING se utiliza en Javascript para acceder a elementos específicos.

¿Qué es htmlCollections y NodeList y que se recomienda hacer con ellos?

Son tipos de objetos que se utilizan en Javascript para mostrar colecciones de elementos del DOM de un sitio web. Ambos son para almacenar listas de nodos HTML.
HTML Collection: Colección de elementos HTML, y se actualiza automáticamente cuando se modifican elementos del DOM. Se utiliza la propiedad children. También, mediante un array.from, se puede acceder a los índices. Se recomienda utilizarlo cuando se trabajo con elementos específicos del HTML, y cuando necesitamos que una colección se actualice automáticamente al cambiar el DOM.
Nodelist: Colección estática de nodos que no se actualiza cuando cambia el DOM. Se utilizan Child Nodes o querySelectorAll como métodos. También, se puede acceder mediante índices e iterando entre ellos. Este objeto es más recomendable cuando usamos nodos, y cuando necesitamos que la lista sea estática.

Se refiere a la acción que ocurre en el programa, y como este detecta y responde a la misma. Son esenciales para la interacción de los usuarios en sitios web. Las acciones más comunes son: hacer click en un sition web, scrollear, o presionar teclas.

Definición de evento click y ejemplo

El evento "click" es muy común en programación, y se centra en la acción de hacer click sobre un elemento en específico del sitio web. Al hacer click, se activan funciones que están relacionadas con el evento. Son muy utilizados en la programación para generar interacciones más dinámicas por parte del usuario al utilizar el sitio web. Se utiliza la función addEventListener, y con ello se define el comportamiento que deberá generar una vez el usuario haga click en el elemento.

Definicion de vento scroll y ejemplo

Scroll, que significa desplazamiento, hace referencia a un evento que se genera cuando un usuario realiza un desplazamiento en el sitio web, ya sea horizontal o vertical. Permite detectar cuando el usuario está navegando y permite crear efectos visuales según el desplazamiento del usuario alrededor del sitio. Se recomienda utilizar window al añadir el elemento para que el scroll se aplique a todo el sitio.
